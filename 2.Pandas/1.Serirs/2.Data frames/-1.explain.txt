| **Method**          | **Description**                                                                 |
|---------------------|---------------------------------------------------------------------------------|
| `head()`            | Returns the first n rows (default 5) of the DataFrame.                           |
| `tail()`            | Returns the last n rows (default 5) of the DataFrame.                            |
| `info()`            | Provides a concise summary of the DataFrame, including data types and memory.    |
| `describe()`        | Generates descriptive statistics for numerical columns.                         |
| `columns`           | Returns the column labels of the DataFrame.                                      |
| `shape`             | Returns the dimensionality of the DataFrame (rows, columns).                     |
| `dtypes`            | Returns the data types of each column.                                           |
| `loc[]`             | Accesses a group of rows and columns by labels.                                  |
| `iloc[]`            | Accesses rows and columns by integer-location based indexing.                    |
| `drop()`            | Removes specified labels from rows or columns.                                   |
| `sort_values()`     | Sorts the DataFrame by the values of a specified column.                         |
| `groupby()`         | Groups data and applies a function to each group (e.g., sum, mean).              |
| `apply()`           | Applies a function along an axis of the DataFrame.                               |
| `isnull()`          | Detects missing values in the DataFrame.                                         |
| `fillna()`          | Fills missing values with a specified value.                                     |
| `merge()`           | Merges two DataFrames on a key or index.                                         |
| `concat()`          | Concatenates two or more DataFrames along a particular axis (row-wise or col-wise).|
| `pivot()`           | Reshapes data based on column values.                                             |
| `reset_index()`     | Resets the index of the DataFrame.                                               |
| `set_index()`       | Sets a specific column as the index of the DataFrame.                            |
